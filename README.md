# Sentiment Analysis System

A complete sentiment analysis system built with Python, scikit-learn, and Flask.

## Features

- Text preprocessing (cleaning, tokenization, stopword removal, stemming)
- Machine learning model using TF-IDF and Logistic Regression
- RESTful API for sentiment prediction
- Both single and batch text analysis
- Easy to extend and customize
